# プロジェクト名：Sasaki's Portfolio（ポートフォリオサイト）

## Overview
「Clarity, Structure, The Human Centric Code（明快な構造、そして人に寄り添うコード）」をテーマに、自分自身のスキルと想いを詰め込んだポートフォリオサイトです。
保育士からエンジニアへの挑戦という背景を大切に、使う人に寄り添う「優しさ」と、エンジニアとしての「論理的な設計」の両立を目指しました。

## Technical Features & Focus

### 1. 視線を導く「一本のライン」のアニメーション
スクロールに合わせて垂直なラインが伸びていく演出を取り入れました。これは「一本の筋が通った誠実な仕事」を表現すると同時に、読み手の視線を自然に次のコンテンツへと導く役割を持たせています。

### 2. 修正や拡張に強いコーディング
将来の作品追加やデザイン変更をスムーズに行えるよう、クラス名の付け方（BEM設計）や、色・サイズの一括管理（CSS変数）を徹底しました。チーム開発でも「読みやすく、管理しやすい」と言われるような、堅実なコードを記述しています。

### 3. あらゆるデバイスで崩れないレイアウト
スマホから大きなモニターまで、どの画面で見ても文字の大きさや余白が心地よいバランスに保たれるよう設計しました。フロントエンドエンジニアとして、デザインを再現しつつ、使い心地（UX）を高める工夫を凝らしています。

## Tech Stack
- **Languages:** HTML5 / CSS3 / JavaScript (Vanilla JS)
- **Key Techniques:** BEM設計 / レスポンシブ設計（clamp関数） / Intersection Observer API
- **Design Tools:** Figma / Adobe Photoshop

## Production Notes
このサイトの設計図（Figma）から実装まで、すべて自ら手がけました。保育士時代に培った「相手を観察し、必要なものを先回りして用意する」というホスピタリティを、コードとデザインの両面から表現しています。
