# プロジェクト名：Patisserie Le nez de la Patisserie
## Overview
「記憶に残る香り」をテーマに、手作りケーキの温かみと洗練されたブランドイメージを伝えるコンセプトサイトです。
単に視覚的な美しさを追及するだけでなく、焼き上がりの香りや素材の風味まで想像させるようなビジュアル設計を行い、ユーザーがストレスなく情報を探せる実用性と両立させました。

## Technical Features & Focus

### 1. 「香り」を連想させるビジュアル・プログラミング
高解像度な素材写真や、湯気・質感を感じさせるカットを戦略的に配置。画像形式の最適化や遅延読み込み（Lazy Loading）を実装することで、読み込み速度を犠牲にすることなく、香りを連想させる豊かなブラウジング体験を実現しました。

### 2. 世界観を補完するタイポグラフィと余白
ブランドの「気品」と、オーブンから漂う「優しさ」を表現するため、セリフ体とサンセリフ体を効果的に使い分けました。clamp()関数によるレスポンシブな余白管理により、どのデバイスでも店舗のショーケースを眺めているような、整然かつ温かみのあるレイアウトを維持します。

### 3. 直感的なインフォメーション設計（The Human Centric Code）
店舗情報や、香りにこだわった季節限定商品のラインナップなど、ユーザーが求める情報へスムーズに到達できる導線を構築。アクセシビリティを意識したカラーコントラストを設定し、幅広い層の顧客にとって「使いやすく、記憶に残る」サイトを目指しました。

## Tech Stack
- **Languages:** HTML5 / CSS3 / JavaScript
- **Key Techniques:** Responsive Design / Flexbox & Grid Layout / Liquid Design
- **Design:** Adobe Photoshop (Self-directed/Concept design)

## Production Notes
「香りが届くようなデザイン」という抽象的なテーマを、エンジニアリングの側面から具体化しました。BEM設計によるコンポーネント管理を行い、季節ごとの商品入れ替えや更新が容易な保守性の高い構造を構築することで、常に「鮮度の高い」情報発信を可能にしています。
