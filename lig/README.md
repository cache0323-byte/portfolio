# プロジェクト名：デジタルハリウッドSTUDIO by LIG（スクール紹介サイト）

## Overview
Webクリエイティブスクール「デジタルハリウッドSTUDIO by LIG」の紹介サイトです。
「新しい自分に出会えるワクワク感」をテーマに、スクールの活気あふれる雰囲気と、安心して学べる信頼感が伝わるサイトを目指しました。

## Technical Features & Focus

### 1. 活気を感じさせる「止まらないスライダー」
スクールの賑やかさや進化し続ける様子を表現するために、写真がずっと横に流れ続けるスライダーを作りました。途中で途切れることなくスムーズに動かすことで、見ていてワクワクするような楽しさを演出しています。

### 2. どんなスマホでも「スルスル」動く快適さ
動きのあるサイトは重くなりがちですが、古いスマホや電波の弱い場所でも動きがカクカクしないように、プログラムの書き方を工夫しました。いつでも、どこでも「心地よく動く」ことにこだわっています。

### 3. 次の一歩を応援する「親切な案内」
サイトを見た人が迷わず次のアクション（説明会予約など）に進めるよう、ボタンの位置や色を調整しました。ボタンに触れた時の反応を少し加えるなど、使う人に寄り添った「迷わせない工夫」を散りばめています。

## Tech Stack
- **Languages:** HTML5 / CSS3 / JavaScript
- **Key Techniques:** スムーズなアニメーション / スマホ対応（レスポンシブ）
- **Design:** Figma / Adobe Photoshop

## Production Notes
Figmaで作ったデザインを元に、ズレや文字の隙間まで丁寧に作り込みました。単に見た目を真似するだけでなく、実際に使う人が「使いやすい」と感じる工夫をエンジニアの視点でプラスしています。
