# プロジェクト名：まつエクサロン Rissyu（紹介サイト）

## Overview
落ち着いた空間で施術を行う、まつエクサロン「Rissyu」の紹介サイトです。
サロンのこだわりである「静かでリラックスできる時間」を大切にし、初めての方でも安心して予約できるような、上品で親しみやすいサイトを目指しました。

## Technical Features & Focus

### 1. ふわっと現れる優しいアニメーション
スクロールに合わせて、画像や文字が自然に浮き上がってくるような動きを作りました。サロンのゆったりした雰囲気を感じてもらえるよう、あえてゆっくりと動かすことで「安らぎ」を表現しています。

### 2. 画面を引き締める「縦のライン」
デザインのアクセントとして、細い縦のラインをサイト全体に配置しました。このラインがあることで、余白がすっきりして見え、サロンの「清潔感」や「凛とした上品さ」が伝わるように工夫しています。

### 3. 知りたい情報がすぐに見つかる工夫
場所、営業時間、メニューなど、お客様が知りたい情報をシンプルに配置しました。スマートフォンでも文字が読みやすい大きさに自動で調整されるように設定し、どなたでも迷わず使える「親切な設計」を心がけています。

## Tech Stack
- **Languages:** HTML5 / CSS3 / JavaScript
- **Key Techniques:** レスポンシブ設計（スマホ・PC対応） / スクロールアニメーション
- **Design:** Figma / Adobe Photoshop

## Production Notes
サロンの「凛とした空気感」を形にするために、文字の隙間や画像の配置を調整しました。保育士の経験で大切にしてきた「相手の立場に立つ」という想いを、使いやすさと美しさの両立という形で表現しています。
